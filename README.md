# Atio Studio - Quiet Premium Landing Page

Atio Studio(아티오 스튜디오)는 비즈니스의 본질을 꿰뚫는 고감도 브랜딩과 웹 개발 서비스를 제공하는 전문 에이전시의 랜딩 페이지입니다. **"Quiet Premium"** 디자인 언어를 바탕으로, 절제된 화려함과 압도적인 사용자 경험을 전달합니다.

## 🚀 주요 특징 (Key Features)

- **Quiet Premium Design**: 블랙과 화이트의 강렬한 대비, 세련된 타이포그래피, 그리고 미세한 그레인(Grain) 질감을 활용한 프리미엄 디자인.
- **Interactive Visuals**: Three.js 셰이더를 활용한 에테리얼 배경과 Framer Motion을 이용한 정교한 스태거(Stagger) 애니메이션.
- **Portfolio Masonry**: 비대칭 그리드와 커서 팔로워 인터랙션을 결합한 세련된 작업물 쇼케이스.
- **Conversion-Focused Structure**: 고객의 신뢰를 얻기 위한 PAS 프레임워크(문제-공감-해결) 기반의 섹션 구성 및 100% 환불 보장 섹션.
- **Responsive Layout**: 모바일과 데스크탑 환경 모두에서 최적화된 시각적 경험을 제공하는 완벽한 반응형 설계.

## 🛠 사용 기술 (Tech Stack)

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Background Effects**: GLSL Shaders (ShaderGradient)
- **Language**: TypeScript
- **Deployment**: Netlify (ready)

## 📁 프로젝트 구조

- `/app`: 메인 페이지 및 레이아웃 구성
- `/components/main`: 히어로, 포트폴리오, 프로세스, FAQ 등 주요 섹션 컴포넌트
- `/components/ui`: 질감 오버레이, 네비게이션 등 UI 요소
- `/public`: 고해상도 프로젝트 이미지 및 브랜드 에셋

## ⚙️ 시작하기

1. 의존성 설치:
   ```bash
   npm install
   ```

2. 개발 서버 실행:
   ```bash
   npm run dev
   ```

3. 빌드 및 배포 준비:
   ```bash
   npm run build
   ```
